# giteki

$ giteki help
NAME:
   giteki - A tool for Japan certified radio equipment

USAGE:
   giteki [global options] command [command options] [arguments...]

VERSION:
   0.0.1

AUTHOR(S): 
   daisuzu <daisuzu@gmail.com>

   download	Downloads the xls of Japan certified radio equipment list

   help, h	Shows a list of commands or help for one command
   
GLOBAL OPTIONS:
   --help, -h		show help
   --version, -v	print the version
